Tips - Before modifing T4 Templates

The T4 templates are located in the C:\Program Files (x86)\LayerCake Generator vX.X.X\Templates\ directory.

If you plan to modify them in order to suit your needs it is recommanded to copy the Templates\ directory into your project.

Copy and paste the Templates\ directory into your project

D:\DEV\Com.Example.Labs\Templates\

Open the LayerCake Generator configuration file of the project

D:\DEV\Com.Example.Labs\Com.Example.Labs.config

Set the Config.AdvancedOptions.TemplateRootPath key with the new path

"Config.AdvancedOptions.TemplateRootPath" : "D:\DEV\Com.Example.Labs\Templates\"
