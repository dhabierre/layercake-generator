HowTo - LayerCake Generator CommandLine.

It is possible to execute LayerCake Generator from command-line (located to the install folder).

  • Full process

    LayerCake.Generator.Cmd.exe [Debug|Release] <ConfigFilePath>
    LayerCake.Generator.Cmd.exe Debug "C:\...\Com.Company.Project\Com.Company.Project.config"

  • Process with or without the SQL Procedure Integration

    LayerCake.Generator.Cmd.exe [Debug|Release] <ConfigFilePath> <WithSqlProcedureIntegration>
    LayerCake.Generator.Cmd.exe Debug "C:\...\Com.Company.MyProject\Com.Company.Project.config" true
    LayerCake.Generator.Cmd.exe Debug "C:\...\Com.Company.MyProject\Com.Company.Project.config" false

  • Process with or without the SQL Procedure Integration and a set of tables

    LayerCake.Generator.Cmd.exe [Debug|Release] <ConfigFilePath> <WithSqlProcedureIntegration> <Table1> <Table2> ...
    LayerCake.Generator.Cmd.exe Debug "C:\...\Com.Company.Project\Com.Company.Project.config" true User UserRole Role
    LayerCake.Generator.Cmd.exe Debug "C:\...\Com.Company.Project\Com.Company.Project.config" false User UserRole Role
